# Job Engine 🚀

A student job portal with preparation tools and interview support.

## Features
- Dashboard with user data
- Job categories (Campus, Remote, Internships)
- Resume Builder
- Mock Interview
- GD Topics & HR Questions

## Tech Stack
- React.js
- Firebase (Auth + Firestore)
